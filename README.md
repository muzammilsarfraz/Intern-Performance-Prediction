# Intern Performance Prediction Model

## Project Overview
This project predicts an intern's performance score using Machine Learning. The model is trained using attendance, feedback rating, and task completion time.

## Objective
To build a machine learning model that predicts intern performance based on:
- Attendance
- Feedback Rating
- Task Completion Time

## Dataset
The dataset contains the following features:
- Attendance
- Feedback_Rating
- Task_Completion_Time
- Performance_Score (Target)

## Machine Learning Algorithm
- Random Forest Regressor

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Joblib

## Project Workflow
1. Import required libraries
2. Load the dataset
3. Split data into training and testing sets
4. Train the Random Forest Regression model
5. Predict intern performance
6. Evaluate the model using MAE, MSE, and R² Score
7. Save the trained model

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score
